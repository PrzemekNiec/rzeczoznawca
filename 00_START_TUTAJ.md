## 🚀 Dashboard Rzeczoznawcy – Pierwsze Uruchomienie

Witaj w Twoim nowym centrum dowodzenia! Ta aplikacja została stworzona, aby zaoszczędzić Twój czas i wyeliminować żmudne przeklejanie numerów działek między dziesiątkami kart przeglądarki.

### 1. Przygotowanie techniczne (Tylko raz)

- **Dysk Google:** Upewnij się, że aplikacja _Google Drive for Desktop_ jest uruchomiona, a Twój sejf Obsidian jest zsynchronizowany.
    
- **Link do aplikacji:** Wejdź na adres: `https://narzedzia.hipoteki.eu` (lub inny adres Firebase, który poda Przemek).
    

### 2. Konfiguracja "Mocy" Dashboardu

Przy pierwszym uruchomieniu musisz nadać aplikacji dwa kluczowe uprawnienia:

- **Odblokowanie Pop-upów:** Gdy spróbujesz otworzyć wiele kart naraz, przeglądarka może to zablokować. Kliknij ikonę „zablokowanego okna” w pasku adresu i wybierz: **„Zawsze zezwalaj na wyskakujące okienka”**.
    
- **Dostęp do Sejfu:** Kliknij przycisk „Połącz z Obsidianem” i wybierz folder swojego sejfu. Dzięki temu aplikacja będzie czytać Twoje własne linki do gmin.
    

### 3. Twój codzienny Workflow

1. **Wklej numer:** Wpisz lub wklej numer działki/TERYT w główne pole wyszukiwania.
    
2. **Sprawdź podgląd:** Dashboard od razu pokaże Ci, jaką gminę i obręb rozpoznał.
    
3. **Uruchom Analizę:** Kliknij przycisk **"Otwórz źródła"** lub użyj skrótu `Alt + O`.
    
4. **Kopiuj-Wklej:** Używaj ikonek „kopiuj” obok numerów, aby błyskawicznie przenosić czyste dane do operatu w Wordzie.
    

### 4. Najważniejsze skróty (Oszczędność czasu)

| Skrót     | Akcja                                      |
| --------- | ------------------------------------------ |
| `Enter`   | Zatwierdź numer działki.                   |
| `Alt + O` | Otwórz wszystkie mapy i bazy (Multi-Open). |
| `Esc`     | Wyczyść wyszukiwarkę.                      |
[!TIP] Wskazówka Jeśli znajdziesz nowy, lepszy link do MPZP w jakiejś gminie, po prostu dopisz go do swojej tabeli w pliku `Linkownik_Gminny.md` w Obsidianie. Dashboard sam go "zauważy" przy następnym odświeżeniu!
