# Główne Źródła Danych (Krajowe)

| Nazwa Systemu | Adas URL | Opis / Cel |
| :--- | :--- | :--- |
| **Geoportal Krajowy** | `mapy.geoportal.gov.pl` | Podstawowa mapa, granice działek, numery TERYT. |
| **EKW (Księgi Wieczyste)** | `ekw.ms.gov.pl` | Stan prawny nieruchomości (wymaga numeru KW). |
| **GUNB (Wyszukiwarka)** | `wyszukiwarka.gunb.gov.pl` | Rejestr wniosków i decyzji o pozwolenie na budowę. |
| **Mapa Akustyczna** | `architektura.um.warszawa.pl` | (Przykład dla miast) Hałas drogowy i lotniczy. |
| **Zabytek.pl** | `zabytek.pl` | Sprawdzenie czy obiekt nie jest w rejestrze zabytków. |
| **Polska pod Lupą** | `polskapodlupa.pl` | Szybki podgląd parametrów działki. |

## Parametryzacja Linków
Aplikacja powinna automatycznie wstrzykiwać TERYT lub numer działki w odpowiednie miejsca w URL tych serwisów (o ile system na to pozwala).
