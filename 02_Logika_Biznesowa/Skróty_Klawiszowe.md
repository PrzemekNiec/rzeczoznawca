# Skróty Klawiszowe Aplikacji

| Skrót     | Akcja                                              |
| :-------- | :------------------------------------------------- |
| `Enter`   | Uruchom parsujący "SmartSearch".                   |
| `Alt + O` | Otwórz wszystkie zdefiniowane źródła (Multi-Open). |
| `Alt + C` | Skopiuj czysty numer TERYT do schowka.             |
| `Alt + H` | Przejdź do pola wyszukiwania (Focus Search).       |
| `Esc`     | Wyczyść pole wyszukiwania.                         |
